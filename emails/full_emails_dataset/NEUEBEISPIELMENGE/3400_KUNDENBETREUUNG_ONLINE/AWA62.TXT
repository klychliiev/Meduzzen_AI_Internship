Sehr geehrter Herr Täge


vielen Dank für Ihre Nachricht.

Gern haben wir Ihre Kundendaten überprüft und testweise mit Ihrer Kunden-
Rufnummer eine Registrierung durchgeführt. Das System steht 
uneingeschränkt zur
Verfügung. Nachfolgend haben wir die notwendigen Registrierungsschritte
aufgeführt Falls auf Ihrem PC bereits eine Internetverbindung installiert
ist
können Sie bei bestehender Internetverbindung bei Schritt 6 beginnen.

Schritt1:
Doppelklicken Sie das Symbol  Arbeitsplatz.
Doppelklicken Sie auf DFÜ-Netzwerk. Sollte dieser Ordner bei Ihnen
nicht vorhanden sein
müssen Sie das DFÜ-Netzwerk von Ihrer Original
Windows 95/98-CD nachinstallieren. Wie das geht
entnehmen Sie bitte der
Anleitung Ihres Betriebssystems.
Doppelklicken Sie auf Neue Verbindung erstellen.

Schritt 2:
Geben Sie den Namen der neuen Internetverbindung ein
(E-Plus Online-Registrierungsdienst).
Wählen Sie ein Modem oder ISDN-Karte.
Klicken Sie auf den Weiter  Button.

Schritt 3:
Geben Sie die Rufnummer 0 19 21 93 des E-Plus Registrierungsdienstes ein
Klicken Sie auf Weiter und dann auf Fertigstellen. Im DFÜ-Netzwerk ist
nun ein neues Symbol eingerichtet worden.

Schritt 4:
Klicken Sie mit der rechten Maustaste auf das Symbol
E-Plus Online-Registrierungsdienst.
Wählen sie den Unterpunkt Eigenschaften
Klicken Sie auf die Kategorie Servertypen.
Aktivieren Sie nur das Kästchen TCP / IP.
Bestätigen Sie mit O.K..

Schritt 5:
Doppelklicken Sie auf das Symbol E-Plus Online-Registrierungsdienst.
Tragen Sie als Benutzername eplus@eplus-online.de ein.
Tragen Sie als Kennwort online ein.
Klicken Sie auf Verbinden. Wurde die Verbindung erfolgreich aufgebaut

erscheint unten rechts auf Ihrem Bildschirm ein Verbindungssymbol (2 PCs).
Wenn Sie die Verbindung später wieder trennen möchten
dann klicken Sie
mit der rechten Maustaste auf dieses Symbol und wählen dann die Option
Trennen.

Schritt 6:
Starten Sie Ihren Browser
und geben Sie http://www.eplus.de als
Adresse ein. Sie gelangen so zum Portal von E-Plus.
Klicken Sie auf Werden Sie E-Plus Online Kunde - .... So gelangen Sie zur
Anmeldung für E-Plus Online.
Folgen Sie der Anmeldung

WICHTIGER HINWEIS:
Erst nachdem Sie die Anmeldung auf dem Portal durchlaufen haben und hier
Ihr individuelles Login (Benutzername) und PIN (Kennwort) erhalten haben

können Sie die günstige Einwahlnummer 0 19 21 92 nutzen und kommen in
den Genuß vieler weiterer Vorteile
wie z.B. 60 Freiminuten für
